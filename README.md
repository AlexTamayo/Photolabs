## SETUP

Install dependencies with `npm install` in `/frontend` and `/backend`, respectively.

### [Frontend] Running Webpack Development Server

```sh
cd frontend
npm start
```

### [Backend] Running Backend Servier

Read `backend/readme` for further setup details.

```sh
cd backend
npm start
```
